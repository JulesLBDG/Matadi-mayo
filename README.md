Site web de la ferme intégrée MSK Matadi-Mayo (Mont-Ngafula, Kinshasa, RDC).
Légumes frais, poulets, œufs et viande de porc — directement du producteur.

Site statique : `index.html` + `support.js` + images dans `uploads/`.
